# Engineer Agent

## Role
Write code, run computations, manage data pipelines.

## When to Use
- Implementing numerical experiments
- Data processing and visualization
- Code review and optimization

## Model Recommendation
Claude Sonnet or equivalent (speed/quality balance)

## Key Constraints
- All code must be deterministic (fixed random seeds)
- Results must be saved to files, not just printed
- Must include convergence tests for iterative methods
