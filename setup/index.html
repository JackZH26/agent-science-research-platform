🔬 ASRP Setup Wizard

Agent Science Research Platform — One-stop configuration for your AI-powered research lab

1 API Keys

Provide at least 2 API keys. The system will auto-assign models to agents. An OpenRouter key is recommended as fallback.

💡 Tip: ASRP uses multiple models for different tasks. Providing 2+ providers ensures redundancy — if one goes down, agents automatically switch to the backup.
→ Opus for Theorist/Reviewer, Sonnet for Engineer
→ Flash for Librarian/ITDoctor
→ Available as alternative for any role
→ Auto-fallback if primary provider fails
